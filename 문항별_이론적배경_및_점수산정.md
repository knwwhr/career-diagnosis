| 항목 | 이론적 배경 | 측정 내용 | 점수 영향 | 계산 방식 |
|------|-------------|-----------|-----------|-----------|

**측정 가치관 목록:**
- 안정성: 예측 가능한 환경에서 일하는 것
- 성장: 지속적으로 배우고 발전하는 것  
- 창의성: 새로운 아이디어를 만들어내는 것
- 자유로움: 자율적으로 일정과 방식을 결정하는 것
- 의미있는 일: 사회에 도움이 되는 가치있는 일을 하는 것
- 높은 수입: 경제적으로 여유로운 생활을 하는 것
- 워라밸: 일과 개인 생활의 균형을 맞추는 것

**RIASEC 연결 가중치:**
| 가치관 | R | I | A | S | E | C |
|--------|---|---|---|---|---|---|
| 안정성 | 2 | 1 | 0 | 1 | 1 | 3 |
| 성장 | 1 | 3 | 2 | 2 | 3 | 1 |
| 창의성 | 0 | 2 | 3 | 1 | 2 | 0 |
| 자유로움 | 2 | 2 | 3 | 1 | 2 | 0 |
| 의미있는 일 | 1 | 2 | 2 | 3 | 1 | 1 |
| 높은 수입 | 1 | 1 | 1 | 0 | 3 | 2 |

### 1.2 업무환경 선호도 (work_environment)

| 항목 | 이론적 배경 | 측정 내용 | 점수 영향 | 척도 설명 |
|------|-------------|-----------|-----------|-----------|
| **기본 개념** | 환경심리학<br>조직행동학 물리적 환경 이론 | 선호하는 업무 환경 특성 | **최대 15점** | 5점 척도 (별로→정말 좋아요) |

**측정 환경 요소:**
1. **실내 근무**: 사무실이나 실내에서 근무하는 것
2. **유연한 시간**: 출퇴근 시간이 자유로운 것  
3. **팀 협업**: 동료들과 함께 협업하는 것
4. **창의적 업무**: 새로운 아이디어가 필요한 업무
5. **도전적 업무**: 어려운 문제를 해결하는 업무

**점수 계산 방식:**
```
환경 일치도 점수 = 각 환경 요소별 일치도 합산
- 완전 일치 (차이 0): 3점
- 거의 일치 (차이 1): 2점  
- 부분 일치 (차이 2): 1점
- 불일치 (차이 3+): 0점
최대 3점 × 5개 요소 = 15점
```

## 📈 최종 점수 산정 알고리즘

### 직업별 점수 계산 공식
```
최종 점수 = 기본점수(10) + RIASEC매칭(0-30) + 관심산업(0-20) + 스킬자신감(0-25) 
           + 직무이해도(0-20) + 가치관일치(0-8) + 업무환경일치(0-15) + 경험페널티(0 or -10)

최대 가능 점수: 95점 (페널티 없는 경우)
최소 가능 점수: 25점 (페널티 있는 경우)
```

### 결과 해석 기준
| 점수 구간 | 적합도 | 해석 |
|-----------|---------|------|
| 80-95점 | 매우 높음 | 개인 성향과 매우 잘 맞는 직업 |
| 70-79점 | 높음 | 개인 성향과 잘 맞는 직업 |
| 60-69점 | 보통 | 고려해볼 만한 직업 |
| 50-59점 | 낮음 | 추가 검토가 필요한 직업 |
| 50점 미만 | 매우 낮음 | 현재 성향과 잘 맞지 않는 직업 |

### 추천 직업 선정 기준
- **상위 5개 직업**: 가장 높은 점수 순으로 추천
- **점수 차별화**: 개인별로 25-95점 범위에서 차별화된 점수 산출
- **다양성 보장**: 서로 다른 직업군에서 고르게 추천

### RIASEC 차트 연동
- **데이터 소스**: 성향 질문(3점) + 가치관 가중치(최대 9점) = 각 영역별 최대 12점
- **시각화**: Chart.js 레이더 차트, 6각형 형태
- **척도**: 20점 만점으로 정규화하여 표시

### 추천 직무 연동  
- **상위 5개 직업**: 점수 내림차순 정렬
- **적합도 표시**: 95점 만점 기준 백분율 표시
- **근거 제공**: 각 직업별 점수 기여 요소 설명
- **신뢰성 표시**: 정부 출처 데이터가 있는 직업만 성장전망/연봉 정보 표시

### 실행 계획 개선사항
- **기존**: "좀 더 자세히 알아보기" 외부 링크 버튼
- **개선**: 실용적인 조언 (practicalTip) 제공
  - 이력서: "채용공고 3-5개를 먼저 분석하고, 그에 맞는 키워드를 이력서에 포함하세요"
  - 포트폴리오: "질보다 양으로 시작하세요. 작은 프로젝트 3개가 큰 프로젝트 1개보다 나을 수 있습니다"
  - 스킬별 맞춤 조언: 코딩, 분석, 커뮤니케이션 등 9개 스킬별 구체적 실행 팁

### 향후 개선 방향성
1. **전공 연동 시스템**: 교육 배경 질문 추가로 전공-직업 매칭 보너스 적용
2. **개인화 강화**: 사용자 응답 패턴 기반 추천 정밀도 향상  
3. **실시간 피드백**: 질문 응답 과정에서 즉각적인 가이드 제공
