# AI 커리어 진단 도구 🚀

> **배포 완료!** 🎉 https://knwwhr.github.io/career-diagnosis

취업준비생을 위한 **3단계 개인 맞춤형 커리어 진단** 서비스입니다. RIASEC 성향 분석을 통해 56개 직업 중 최적의 직무를 추천하고, 개인화된 액션플랜을 제공합니다.

## ✨ 주요 특징

- **⏱️ 10분 완성**: 4분 + 3분 + 3분의 효율적인 진단 프로세스
- **📊 과학적 분석**: Holland의 RIASEC 직업흥미 이론 기반
- **🎯 56개 직업 추천**: 9개 주요 직군의 다양한 직업 데이터베이스
- **📱 완전 반응형**: 모바일 최적화된 터치 인터페이스
- **🎨 직관적 UI**: 세로 프로세스 배열, 간편한 네비게이션

## 🎯 진단 프로세스

### 1단계: 강점 발견 (4분)
- **가치관 우선순위**: Schwartz의 기본 인간 가치 이론 기반
- **업무환경 선호도**: 환경심리학 이론 적용
- **RIASEC 성향 분석**: Holland의 직업흥미 이론 (1997)
- **강점 경험**: Clifton StrengthsFinder 기반 역량 분석

### 2단계: 직무 매칭 (3분)  
- **관심 산업**: 10개 주요 산업 분야 매칭
- **직무 이해도**: 6개 핵심 직무에 대한 이해 수준
- **스킬 자신감**: Bandura의 자기효능감 이론 적용

### 3단계: 실행 계획 (3분)
- **취업 일정**: 목표 설정 이론 기반
- **준비 상황**: 취업 준비도 평가 모델
- **학습 선호도**: 개인 맞춤형 학습 방법 추천

## 📊 점수 계산 시스템

### 직업 매칭 알고리즘 (최대 95점)
1. **기본 점수**: 10점
2. **RIASEC 성향 매칭**: 최대 30점 (핵심 가중치)
3. **관심 산업 일치**: 최대 20점
4. **스킬 자신감**: 최대 25점
5. **직무 이해도**: 최대 20점
6. **가치관 일치도**: 최대 8점
7. **업무환경 일치도**: 최대 15점
8. **경험 부족 페널티**: -10점

### RIASEC 성향 분석
- **R (현실형)**: 기계/도구를 다루는 실무적 작업
- **I (탐구형)**: 문제를 분석하고 해결하는 연구 활동
- **A (예술형)**: 예술적/창의적 표현 활동
- **S (사회형)**: 사람을 돕고 소통하는 활동
- **E (진취형)**: 리더십/사업을 추진하는 활동
- **C (관습형)**: 체계적으로 정리하고 관리하는 활동

## 🛠 기술 스택

### Frontend
- **HTML5**: 시맨틱 마크업, 접근성 최적화
- **CSS3**: Flexbox/Grid, 반응형 디자인, 모바일 최적화
- **Vanilla JavaScript**: ES6+ 모듈 패턴
- **Chart.js**: RIASEC 레이더 차트 시각화
- **LocalStorage**: 진행상황 자동 저장

### 아키텍처
- **모듈화 설계**: 기능별 JavaScript 파일 분리
- **API 준비**: 백엔드 연동을 위한 AssessmentAPI 클래스
- **컴포넌트 기반**: 재사용 가능한 UI 컴포넌트

## 📁 파일 구조

```
frontend/
```
